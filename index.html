Human Gait Recognition

Github: human-gait

This is my high school senior research project. I obtained treadmill silhouette videos from the OU-ISIR database and worked on extracting features like stride length and foot strikes. Later on, I wrote some simple classification code to identify subjects based on training video data.

Interment.net Cemetery Analysis

View Interment Data Analysis

This was my final project for my undergraduate "Data Science" class. I was taking "Introduction to Anthropology" at the same time, and we had a lab where we manually copied ~30 data points from interment.net and made some non-rigorous conclusions...so I decided to expand on the project by scraping far more data and performing statistical tests. Not that these results are rigorous either. It's a data science class project.
